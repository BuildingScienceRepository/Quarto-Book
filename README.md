# Building Science Atlas

[![Quarto Publish](https://github.com/BuildingScienceRepository/Quarto-Book/actions/workflows/publish.yml/badge.svg)](https://github.com/BuildingScienceRepository/Quarto-Book/actions/workflows/publish.yml)

A collaborative, GitHub-based repository and knowledge base compiling the current state of sustainability analysis for building science and urban analysis.

This project approaches questions from a systems and complexity perspective—going from simple ballpark estimates to full-scale simulations.

## Contributing

We welcome contributions from students and researchers! This project uses a "docs-as-code" workflow powered by [Quarto](https://quarto.org/). Content lives as Markdown/Notebooks (`.qmd`) in this repository.

To contribute:
1. Fork or branch this repository.
2. Edit the `.qmd` files or add new ones to explore topics from most simple to most complex analysis.
3. Submit a Pull Request.

Once merged, GitHub Actions will automatically render and publish the updated book to GitHub Pages.
